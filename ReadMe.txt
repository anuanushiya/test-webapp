==> This project contains a simple Web Application example with Java / J2EE platform on Tomcat app-server v7.x.xx and MySQL database v5.x.xx
==> URL: http://localhost:<PORT>/TestWebApp/index.jsp  ( for example - http://localhost:8080/TestWebApp/index.jsp )

==> Environment ( used for testing this Web Application example )
    --------------------------------------------------------------
    Java (JDK) : 1.6 / 1.7 with any update
    OS : Windows XP or Windows 7
    Build Tool: apache-ant-1.8.1
    App. Server: Tomcat 7.0.53
    MySQL Server: 5.x

==> Abbriations used in this example:
    ----------------------------------
	db - Database ( Scripts and Data Source )
	dd - Deployment Descriptor
	src - Source ( Codes )
	ui - User Intercafe ( Codes )

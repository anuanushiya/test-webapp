==> This project contains a simple Web Application example with Java / J2EE platform on JBoss4.0.3SP1 as well as on JBoss5.0.1 GA Servers
==> URL: http://localhost:<PORT>/TestWebApp  ( for example - http://localhost:8080/TestWebApp )

==> Environment ( used for testing this Web Application example )
    --------------------------------------------------------------
    Java (JDK) : 1.6 with any update
    OS : Windows XP or Windows 7
    Build Tool: apache-ant-1.8.1
    App. Server: JBoss5.0.1 GA
    MySQL Server: 5.x

==> Abbriations used in this example:
    ----------------------------------
	db - Database ( Scripts and Data Source )
	dd - Deployment Descriptor
	src - Source ( Codes )
	ui - User Intercafe ( Codes )
